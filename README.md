# Refactoring Tennis

## Installation

```
npm install
```

## Tests

```
npm test
```
or `npm run test`

## Watch Tests

```
npm run watch
```
or `npm test -- --watch`

## Coverage

```
npm run cover
```

## Serve Coverage

```
npm run serve
```

**The original tennis logic is a fork from [Emily Bache](https://github.com/emilybache/Tennis-Refactoring-Kata/).**
